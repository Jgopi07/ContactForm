# 📩 Contact Form Web Application

A full-stack contact form application that allows users to submit messages, with secure backend handling and data storage.

## 🔥 Features

* 📝 User-friendly contact form (Name, Email, Message)
* ✅ Input validation (frontend + backend)
* 🔐 Secure data handling
* 💾 Stores submitted data (file / database)
* ⚡ Fast API response using Node.js & Express
* 🎨 Clean and responsive UI

---

## 🧠 How It Works

1. User fills the contact form
2. Frontend sends data to backend (API)
3. Backend validates input
4. Data is stored securely
5. Success response is shown
